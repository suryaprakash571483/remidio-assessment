# Remidio Assessment

A frontend assessment project built with modern web technologies.  
This project demonstrates responsive design, clean UI components, and beginner‑friendly CSS styling.

---

## 🚀 Features
- Responsive navigation bar
- Hero section with call‑to‑action buttons
- Grid‑based layout for sections
- Footer with branding
- Beginner‑friendly CSS (no variables, simplified styles)

---

## 🛠️ Tech Stack
- **HTML5** – Structure of the application
- **CSS3** – Styling and layout
- **JavaScript (ES6)** – Interactivity
- **Node.js** – For package management and development setup

## ⚙️ Installation
Clone the repo and install dependencies:

`
git clone https://github.com/suryaprakash571483/remidio-assessment.git
cd remidio-assessment
npm install`

## Start the development server:
`npm start`
