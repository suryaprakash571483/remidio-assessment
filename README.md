# Oculomics Landing Page

AI-powered retinal health platform landing page built with React.

## Quick Start

```bash
npm install
npm start
```

Opens at http://localhost:3000

## Build for Production

```bash
npm run build
```

## Project Structure

```
src/
├── App.js                  ← Entry component
├── index.js                ← React DOM render
├── index.css               ← Global reset (empty)
└── OculomicsLanding.jsx    ← Main landing page
```

## Tech Stack
- React 18
- CSS-in-JS (template literals)
- Google Fonts: Syne + DM Sans
- No external UI libraries
